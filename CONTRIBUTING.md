# Contributing

Thanks for improving this guide.

## Scope

This repository focuses on:

- OpenClaw installation
- AICodeMirror provider configuration
- Verification and troubleshooting flow

## How to contribute

1. Fork this repository.
2. Create a branch: `feat/your-topic` or `fix/your-topic`.
3. Keep changes focused and reproducible.
4. Update both `README.md` and `README_EN.md` when content changes.
5. Open a Pull Request with:
   - What changed
   - Why it changed
   - How to verify

## Content rules

- Commands must be copy-paste runnable.
- Include OS notes when behavior differs.
- Avoid vendor lock-in claims without evidence.

## Report issues

Use GitHub Issues and include:

- OS and shell
- Exact command
- Error output
- Expected behavior
